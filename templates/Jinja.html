Jinja

  1. Out Put

    {{ "Hello World" }}

  2. String Manipulation

    {{ "Hello World" | lower }}

    {{ "Hello" | length }}

  3. Ternary Conditional

    {% set time = "Good Morning!" if hour > 5 and hour < 12 else ("Good Afternoon!" if hour >= 12 and hour < 17 else "Good Evening!") %} {{time }}

  4. Chaining Filters:

    {{ "HELLO WORLD" | lower | capitalize }} # Output: Hello World

  5. String Concatenation

    {{ "It is" ~ hour ~ " AM/PM" }}

  6. Math Operator

    Sum: 1 + 3 ={{ 1 + 3 }}

    Minus: 1 - 3 ={{ 1 - 3 }}

    Device: 1 / 3 ={{ 1 / 3 }}

    Multiply: 2 * 3 ={{ 2 * 3 }}

  7. Variable declaration

    {% set x = 10 %}

    {{ x }}

  8. If Statement

    {% if hour > 5 and hour < 12 %} Good Morning! {% elif hour >= 12 and hour < 17 %} Good Afternoon! {% elif hour >= 17 %} Good Evening! {% endif %}

  9. For Loop

    {% for number in range(1, 11) %} {{ number }} {% endfor %}

  10. Current Date and Time

    Date: {{ now().strftime("%Y-%m-%d") }}

    Time: {{ now().strftime("%H:%M:%S") }}

  11. Number Conversion

    float to decimal

    {{ 10.50482737 | int }}

    Converts Decimal to float

    {{ 10 | float }}

  12. Currency

    {{ "${:.2f}".format(15.10) }}

  13. Basic Macros

    {% macro sayHello() -%} Hello! Nice to meet you {%- endmacro %}

    Welcome Message

    {{ sayHello() }} How are you today?

  14. Random

    {{ ["hello", "hi there!", "howdy!", "hey there!" ]|random }}

  15. Display Array by index

    {% if fruite %}

    The first element is: {{ fruite[0] }}

    The third element is: {{ fruite[2] }}

    {% else %} Array is empty or undefined. {% endif %}

  16. Boolean

    {% set is_admin = True %} {% set has_permission = False %} {% if is_admin and has_permission %}

    You are an admin with the required permission.

    {% else %}

    You don't have sufficient access.

    {% endif %}

  17. Dictionary Access

    {% set person = {'name': 'Alice', 'age': 30} %}

    {{ person['name'] }} # Output: Alice

  18. Dictionary Iteration

    {% set person = {'name': 'Alice', 'age': 30} %} {% for key, value in person.items() %}

    {{ key }}: {{ value }}

    {% endfor %}

  19. Checking for Empty List

    {% if not fruits %}

    The fruits list is empty.

    {% endif %}
  20. Multiple Variable declaration {% set a, b, c = 'A', 'B', 'C' %} a: {{ a }}, b: {{ b }}, c: {{ c }}

    {{ a }}
